# BLDC Driver – STM32F405RGT6


基于 **STM32F405RGT6** 的高性能无刷直流电机（BLDC）驱动器，集成双路电压采集、CAN 总线接口、双路 SPI 绝对值编码器接口，支持 19V/20A 持续电流输出。适用于机器人关节、小型无人机电调、CNC 及 DIY 项目。

## ✨ 特性

- **主控芯片**：STM32F405RGT6 (Cortex-M4，FPU，1MB Flash，192KB RAM)
- **供电电压**：12V–24V（标称 19V）
- **额定电流**：20A 持续（峰值 30A，<1s，需散热）
- **双路电压采样**：母线电压 + 外部电压（如电池电压），12-bit ADC
- **通信接口**：CAN 2.0A/B（TJA1050收发器），支持 1Mbps
- **编码器输入**：2× SPI 总线（独立片选），兼容 TLE5012B、AS5047P、MT6816 等绝对值编码器
- **PWM 驱动**：6 路互补 PWM（带死区插入），最高 32kHz
- **电流采样**：3 路低侧/高侧电流采样接口（需外置运放）
- **保护功能**：过流、过压、欠压、过热保护
- **调试接口**：SWD（4-pin） + 串口调试输出

## 🚀 使用指南

### 1. 硬件连接

1. **电源**：连接 19V 电源到 `VIN` 和 `GND`，注意正负极。确保电源能提供 20A 以上电流。
2. **电机**：将 BLDC 电机的 U、V、W 三相线连接到驱动器输出端子。
3. **编码器**：根据你的编码器数据手册连接 SPI 引脚（SCK, MISO, MOSI）和对应的片选（CS）。两路编码器可独立连接两个电机。
4. **CAN 总线**：连接 `CAN_H`、`CAN_L` 到总线，并在总线两端各接一个 120Ω 终端电阻。
5. **调试/烧录**：通过 SWD 接口连接 ST-Link 或 J-Link 调试器。

> ⚠️ **安全提醒**：19V/20A 具有较高能量，操作时请避免短路，并确保电源线径 ≥ 1.5mm²。
